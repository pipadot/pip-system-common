# Get Started

## Error Status:

- "-600" : Bad Request
- "-602" : Data Not Found
- "-604" : Database connection error
- "-601" : Not Authorized
- "-603" : Route Not Found
- "-605" : Validation Error

## Error Type

- response_status: number
- message: string
- data : Object | undefinded
